# Docs

This folder is for planning notes, design docs, and implementation phases.

The playable static simulator lives in `../app`.

Start here:

- `Doc-Index.md` maps the docs in this repo.
- `vision.md` holds the product roadmap.
- `Realism-Systems/Realism-Systems-Vision.md` holds the realism north star.
- `Realism-Systems/Realism-Systems-Index.md` routes realism work into phases.
- `Realism-Systems/Future/` holds implementation-ready realism plans.
- `Doc-Log.md` records documentation and planning changes.
- `CHANGELOG.md` records shipped app, config, workflow, and runtime behavior changes.
