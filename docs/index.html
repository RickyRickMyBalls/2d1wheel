Model Assumptions

Current calculations are approximate and intended for comparison, not safety prediction.

  • Torque current, power load, thermal load, and speed ceiling are simplified rails.
  • Battery current and phase current are not separated yet.
  • Kv ceiling approximates voltage duty limit until duty cycle is modeled.
  • Motor temperature uses sustained watts, stator size, and cooling approximation.
  • Scrape recovery uses footpad contact geometry and rider lean direction.
